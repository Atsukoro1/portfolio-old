My Projects

These are the projects I am most proud of:

  • Tiktok clone

    This Tiktok clone I made is pretty interesting. It's a fullstack project with a Nuxt frontend and a Nest.js (Node.js) backend. It uses Neo4J as database and Redis for caching. It has two factor authentication and I plan to make personalised recommendations of videos based on the users interests using Tensorflow.

  • CNVM

    CNVM is a Node.js version manager that is both fast and lightweight. It is written in Rust and works on Windows, Linux, and should work also on MacOS in future (not tested).

  • Netscrape

    Netscape is a powerful command line tool written in C that can sniff for incoming packets on specific wireless interfaces and print them out in Human Readable format. This makes it a valuable tool for network administrators and security professionals.
